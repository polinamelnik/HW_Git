1.Из командной строки перейти в папку использовав команду cd + "папка" .

2. На странице репозитория GitHub через кнопку Code
скопировать URL-адрес репозитория.

3. В командной строке написать git clone и скопированый адресс репозитория.

4. Чтобы проверить обнавления нужно выполнить команду         "git pull" в папке репозитория.


Команды Git:
1. git status - показывает изменения рабочего каталога (удаление, изменение, добавление).

2. git log - выдаст список коммитов в репозитории.

3. git remote -v - покажет то какие репозитории были удалены
